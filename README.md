# TrendMart App

TrendMart App build on React and Typescript

## Getting Started

### Prerequisite

- Node
- npm

### Installing

- git clone the repository
- cd in to repository folder and run `npm install`

# To Start Project

- run `npm run dev`

# To Run Unit Tests

- run `npm run test` (for unit tests)

## Features List

- [x] Initial design
